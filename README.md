Este projeto foi desenvolvido com o objetivo de estudar e praticar o framework Spring Boot, aplicando conceitos fundamentais de desenvolvimento backend com Java, 
incluindo persistência de dados, segurança e boas práticas de arquitetura.

Objetivo do Projeto

Aprender a estrutura e funcionamento do Spring Boot

Implementar persistência de dados com Spring Data JPA

Utilizar PostgreSQL como banco de dados relacional

Aplicar Spring Security para controle de autenticação e autorização

Praticar organização de código em camadas (Controller, Service, Repository)

Tecnologias Utilizadas

Java
Spring Boot
Spring Data JPA
Spring Security
PostgreSQL
Hibernate (ORM)
Maven

Banco de Dados

O projeto utiliza o PostgreSQL como banco de dados relacional, integrado ao Spring Boot por meio do Spring Data JPA e Hibernate, permitindo o mapeamento objeto-relacional (ORM) e a 
persistência de dados de forma eficiente.

Segurança

A aplicação utiliza Spring Security para implementar mecanismos básicos de segurança, como:

Autenticação de usuários

Autorização de acesso a endpoints

Proteção de rotas da aplicação

(Configurações em nível de estudo e aprendizado)
